# Question 1

On vous demande de mettre en place une première version de cette application client-serveur en utilisant les tubes comme outil de communication. Vous considérez un seul client et un serveur.

Mettez en place la structure de votre application : processus, nombre de tubes nécessaires, maintien du tableau des réservations en mémoire, structure des données échangées via les tubes. Expliquez vos choix. Pour simplifier, vous pouvez faire afficher chez le client un menu avec les spectacles associés chacun à un  numéro. Le client transmet alors le numéro du spectacle et non son intitulé. 
Codez votre solution en prenant soin de documenter votre code.
Réfléchissez aux difficultés que soulève le fait d'introduire plusieurs clients.
 Vous devez remettre : le code comment de votre client-serveur et un document word expliquant vos choix et vos réponses aux questions.
